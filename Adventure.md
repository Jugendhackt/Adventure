## Library

[pygame](https://pypi.org/project/pygame/) 1.9.6

[bintrees](https://pypi.org/project/bintrees/) 2.1.0

## How to start

>pip install pygame bintree
>
>python main.py

## Control

W => Jump

A => Left

D => Right

R => Restart

F8 => switch to debug mode
